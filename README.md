## Run Locally

Clone the project

```bash
  git clone https://github.com/EliaTG/Github-search-app.git
```

Go to the project directory

```bash
  cd Github-search-app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
